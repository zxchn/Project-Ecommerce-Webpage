A simple e-commerce frontend website built using HTML, CSS, and JavaScript, with assistance from AI tools for code refinement, layout suggestions, and content optimization. The project features a product gallery, a featured video section, and a personalized user greeting using localStorage.

Created as a demo/portfolio project, it highlights frontend development fundamentals such as structure, styling, and basic interactivity. Product images from Penshoppe are used strictly for educational and non-commercial purposes.

Key Features:

Responsive product gallery layout

Featured promotional video section

LocalStorage-based personalized greeting

AI-assisted code improvement and content refinement

Clean and beginner-friendly code structure
